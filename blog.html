Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of multiple listeners to individual child elements.
This is useful for handling events on dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that occurs on a child element propagates (or "bubbles") up through its parent elements, all the way to the root of the document.
This means that when an event is triggered on an element, it is also triggered on its ancestors unless stopped.
